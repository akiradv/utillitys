Sobre o Utillitys

Utillitys é um conjunto de ferramentas úteis desenvolvido para facilitar tarefas do dia a dia. Criado em 2024, o projeto cresceu e hoje oferece uma variedade de funcionalidades.

Atualmente conta com as seguintes ferramentas (clique para acessar):

  • Página Inicial
  • Calculadora de Idade
  • Calculadora IMC
  • Calculadora Simples
  • Corretor Ortográfico
  • Conversor de Moedas
  • Encurtador de URLs
  • Busca de Livros
  • Gerador de Senhas
  • Gerador de QR Code
  • Gerador de Currículo
  • Relógio Mundial
  • Texto para Voz

Projeto open-source no GitHub.

Tecnologias: HTML5, CSS3, JavaScript puro, APIs gratuitas (Open Library, LanguageTool, TinyURL, Web Speech API, ExchangeRate-API).
